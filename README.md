# 📝 Task Tracker API (Spring Boot)

A simple **Task Tracker REST API** built with **Spring Boot**.  
This project demonstrates how to build a basic backend application that allows users to create, manage and track their daily tasks.

It is designed as a beginner-friendly project to learn:

- Spring Boot fundamentals
- REST API development
- CRUD operations
- Layered architecture (Controller, Service, Repository)
- Database integration

---

# 🚀 Features

- Create new tasks
- View all tasks
- View a task by ID
- Update existing tasks
- Delete tasks
- Track task status (completed / pending)

---

# 🛠 Tech Stack

- **Java 17+**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA**
- **Hibernate**
- **H2 / MySQL (depending on configuration)**
- **Maven**

---


Architecture follows the standard **Spring Boot layered pattern**:

Controller → Service → Repository → Database
